# Ping-Pong-Game
This is a game made with vscode as a replica of the OG pong game.

It's a 2 player game with W and S for Player 1 and Up arrow and Down arrow keys for Player 2

# How To Play

## Controls 

### Player 1 Controls: 

W - Up

S - Down




### Player 2 Controls: 

Up arrow key  - Up

Down arrow key - Down
